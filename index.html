Whatsapp Gateway

Gateway ini menggunakan web multi device anda masih bisa terhubung dengan device lain, whatsapp di hp tidak harus selalu terhubung

Untuk menggunakan Whatsapp Gateway

  1. Pastikan whatsapp anda sudah bisa multi device
  2. Buka Whatsapp di telepon anda
  3. Buka menu Whatsapp web dari setelan.
  4. Scan barcode disamping
  5. Anda bisa meninggalkan halaman ini atau menutupnya
